# **App Name**: CriptoFund

## Core Features:

- Campaign Creation: Allow users to create and manage crowdfunding campaigns, defining goals, timelines, and descriptions.
- Donation Processing: Enable secure donation processing via Ethereum and Bitcoin using smart contracts.
- Metamask Integration: Connect Metamask wallet for seamless transactions.
- User Authentication: Provide a straightforward account creation and login process.
- Campaign Browsing: Offer an intuitive interface for browsing active and completed campaigns.
- Progress Tracking: Display real-time progress and metrics for each crowdfunding campaign.
- Transaction History: Maintain a secure record of donation transactions.
- Contribution Tracking: Donors can track their contributions in real-time and receive Vibe Coins as a token of appreciation.
- AI Donation Assistant: AI tool asks users about their preferred donation causes and monthly budget, then automatically allocates donations each month according to their preferences.

## Style Guidelines:

- Primary color: Strong green (#468B97) to signify trust and stability in financial transactions.
- Background color: Very light white (#E1F0F5) to ensure readability and a clean interface.
- Accent color: Light blur (#FFD700) to highlight key elements such as donation buttons and progress bars, creating visual interest.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern look.
- Use financial and blockchain-related icons, in a simple, modern style.
- Design a clear, intuitive layout that simplifies the donation process.
- Use subtle transitions and loading animations to enhance user experience.
